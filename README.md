# Valheim Mod Guide

A comprehensive guide to Valheim mods used on our server.

## Viewing the Guide

The guide is available as a GitHub Pages site. Visit the `index.html` file to see the Valheim Mod Guide.

## GitHub Pages Setup

To enable GitHub Pages for this repository:

1. Go to **Settings** > **Pages**
2. Under "Build and deployment," ensure:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/` (root)
3. Click **Save**

The site will be published at: `https://emeria.github.io/valheim_not_vegas_readme/`

## Content

The guide includes information about:
- **Epic Loot (v0.12.5)**: RPG-style loot drops and enchanting tables
- **Seasonality**: Dynamic seasons with changing textures and buffs
