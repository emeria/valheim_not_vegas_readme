Adds RPG-style rarity (Magic to Mythic) to drops, enchanting at the forge, and bounty hunting.

Hotkeys

G, H, J for specific Magic Item abilities.

Configuration
  • Enable: Automatically active. Access the "Enchant" tab at any Forge.
  • Customizing: Edit magiceffects.json in the config folder to change stat ranges.
Server Note: Both client and server MUST have the exact same magiceffects.json or the mod will fail to load.

Changes world visuals and player stats based on 4 rotating seasons (Winter, Spring, Summer, Fall).

Hotkeys / Console

seasonality timer_reset in the F5 console (Admins only) to restart the season clock.

Configuration
  • Enable: Textures swap automatically. Ensure "High Quality" textures are enabled in game settings for best look.
  • Settings: Use F1 to change season duration (days) or toggle seasonal buffs (like Summer stamina boost).

Speeds up agriculture by planting and harvesting in a large grid instead of one-by-one.

Hotkeys

Hold Left Shift while planting with the Cultivator or interacting with grown crops.

Configuration
  • Grid Size: Default is 5x5. Change this to 3x3 or 10x10 in the F1 menu under "MassFarming".
  • Stamina: There is a config toggle to "Ignore Stamina" if you find mass planting too exhausting.

Automatically pulls wood, coal, and ore from nearby chests to keep fires, smelters, and kilns running.

Hotkeys

None. It works passively when you are near the structure.

Configuration
  • Range: Default pull range is 10m. Increase this via F1 if your storage is further away.
  • Filters: You can exclude specific items (like Fine Wood) so the mod doesn't accidentally burn your build materials.
Server Note: This mod uses ServerSync. If the admin sets a range of 5m, your client cannot override it to be larger.

Allows you to pick up and move chests and furniture without breaking them or emptying the inventory inside.

Hotkeys

Equip Hammer, point at object, and hold Left Ctrl + Left Click to pick up. Right Click to cancel.

Configuration
  • Restrictions: By default, you cannot move items that provide structural support (like support beams). This can be toggled in F1 settings.
  • Weight: You are usually encumbered while "carrying" a moved object.

Small QoL UI tweaks: adds a clock, weather forecast, and better inventory tooltips.

Hotkeys

Hold Alt while hovering over an item to see the original, un-modded tooltip.

Configuration
  • Placement: Use the F1 menu to drag the Clock and Weather Forecast to different corners of your screen.
  • Colors: Every UI element's font and background transparency is customizable.
