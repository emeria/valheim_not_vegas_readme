How to Add Admins

To grant admin rights, you must add the player's SteamID64 to the server files:

  1. Find the adminlist.txt file (usually in the server's root save folder or AppData/LocalLow/IronGate/Valheim).
  2. Add one SteamID per line.
  3. Restart the server or wait for the next auto-check.
Universal Console Commands

Press F5 to open the console. (Must have -console as a Steam Launch Option).

  • devcommands - Enables cheats (Admin only).
  • debugmode - Creative mode (Press B for free build, K to kill nearby mobs, Z for fly).
  • god - Invincibility.
  • ghost - Enemies ignore you.
Mod Specific Commands
  • Seasonality: seasonality timer_reset - Resets the internal clock to the start of the current season.
  • Epic Loot: epicloot bounties - Lists active bounties.
  • Epic Loot: epicloot givemagicitem [rarity] [item] - Spawn a magic item (e.g. Legendary SwordIron).
  • Automatic Fuel: autofuel reset - Forces the mod to scan for nearby containers again.
