Adds 4 rotating seasons with unique textures, weather patterns, and stamina/stat buffs.

How to Enable

Works automatically. Admins can reset the clock using seasonality timer_reset in F5 console.

How to Configure

Each season has its own .json file in config/shudnal.Seasons. You can change season length (Default 10 days) here.

The following mods provide no gameplay features but are REQUIRED for the others to work:

  • Zen.ModLib: Core logic for all "Zen" branded mods.
  • Jotunn: The main tool used to add new items/features to Valheim.
  • BepInEx: The foundation that allows any mod to run.
